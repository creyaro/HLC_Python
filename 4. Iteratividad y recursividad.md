# Algoritmos Recursivos e Iterativos

## **¬øQu√© es un algoritmo iterativo?**
Un algoritmo iterativo es aquel que resuelve un problema repitiendo una serie de pasos mediante bucles (`for` o `while`).

### **Ejemplo: Factorial Iterativo**
El factorial de un n√∫mero `n` se define como:

```
n! = n √ó (n-1) √ó (n-2) √ó ... √ó 1
```

Ejemplos:
- `3! = 3 √ó 2 √ó 1 = 6`
- `4! = 4 √ó 3 √ó 2 √ó 1 = 24`
- `5! = 5 √ó 4 √ó 3 √ó 2 √ó 1 = 120`

C√≥digo en Python:

```python
 def factorial_iterativo(n):
     resultado = 1
     for i in range(2, n + 1):
         resultado *= i
     return resultado

print(factorial_iterativo(5))  # Resultado: 120
```

üìå **Explicaci√≥n paso a paso**
Si `n = 5`, el bucle se ejecuta as√≠:

1. `resultado = 1`
2. `resultado *= 2` ‚Üí `resultado = 2`
3. `resultado *= 3` ‚Üí `resultado = 6`
4. `resultado *= 4` ‚Üí `resultado = 24`
5. `resultado *= 5` ‚Üí `resultado = 120`

El resultado final es `120`, y el bucle ha ejecutado `n-1` operaciones.

## **¬øQu√© es un algoritmo recursivo?**
Un algoritmo recursivo resuelve un problema dividi√©ndolo en subproblemas m√°s peque√±os, llam√°ndose a s√≠ mismo hasta alcanzar un caso base.

### **Ejemplo: Factorial Recursivo**
Matem√°ticamente, el factorial se puede definir de manera recursiva:

```
n! = n √ó (n-1)!
```

Con el caso base:

```
1! = 1
```

Ejemplo con `n = 5`:

```
5! = 5 √ó 4!
4! = 4 √ó 3!
3! = 3 √ó 2!
2! = 2 √ó 1!
1! = 1
```

C√≥digo en Python:

```python
 def factorial(n):
     if n == 0 or n == 1:  # Caso base
         return 1
     return n * factorial(n - 1)  # Caso recursivo

print(factorial(5))  # Resultado: 120
```


üìå **Explicaci√≥n paso a paso**
Si `n = 5`, las llamadas se ejecutan as√≠:

```
factorial(5) ‚Üí 5 * factorial(4)
factorial(4) ‚Üí 4 * factorial(3)
factorial(3) ‚Üí 3 * factorial(2)
factorial(2) ‚Üí 2 * factorial(1)
factorial(1) ‚Üí 1  (caso base alcanzado)
```

Se resuelven las multiplicaciones en orden inverso:
```
factorial(2) = 2 * 1 = 2
factorial(3) = 3 * 2 = 6
factorial(4) = 4 * 6 = 24
factorial(5) = 5 * 24 = 120
```

Al igual que en el caso iterativo, el resultado final es `120`.

## **Otros ejemplos de Recursi√≥n e Iteraci√≥n**

### Suma de los primeros n n√∫meros naturales

Este algoritmo suma los primeros n n√∫meros naturales. Por ejemplo, ``suma_naturales(4) = 1 + 2 + 3 + 4 = 10``

#### Iterativo 

```python
def suma_naturales_it(n):
    suma = 0
    for i in range(1, n + 1):
        suma += i
    return suma
```

#### Recursivo 

```python
def suma_naturales_rec(n, current=1):
if n == 1:
        return 1
    else:
        return n + suma_naturales_rec(n - 1)
```

## **Cu√°ndo Usar Recursi√≥n vs. Iteraci√≥n**
### **Usar Recursi√≥n cuando:**
- El problema se divide naturalmente en subproblemas m√°s peque√±os (ej. √°rboles binarios, b√∫squeda binaria, merge sort).
- Se necesita una soluci√≥n elegante y concisa.

### **Usar Iteraci√≥n cuando:**
- Se requiere eficiencia y menor uso de memoria.
- El problema no se descompone bien en subproblemas m√°s peque√±os.
- La recursi√≥n lleva a llamadas infinitas (es decir, no se ha definido un caso base).
